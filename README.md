## Account

* Methods usados
POST | GET | PUT | DELETE

  * Function index <br>
        Method:GET <br>
                 <i>Retorna todas as condas existentes<i>
    
                     url:api/conta/home
    
  * Function store <br>
        Method: POST <br>
        <i>Cria uma nova conta<i>
           
                    url:api/conta/cadastro
            
  * Function show <br>
        Method:GET <br>
        <i>Retorna a conta referente ao id<i>
